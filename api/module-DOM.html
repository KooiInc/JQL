(inner) createElementFromHtmlString(htmlStr) → {HTMLElement|Comment}

Source:
  • DOM.js, line 72
Convert a html string to an instance of HTMLElement.
Note: the resulting element is always sanitized using the attrbutes/tags settings. Use DOM.element2DOM to fysically insert/append etc. it into your DOMtree
Parameters:
Name Type Description
htmlStr string The html to convert to HTMLElement e.g. <p id="id" class="someClass">Hello <span style="color: green">world</span></p>
Returns:
the newly created HTMLElement instance or nothing (in case of creating a html comment)

HTMLElement | Comment
