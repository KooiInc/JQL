(inner) HandlerFactory/metaHandler(evt)

Source:
  • ExtensionHelpers.js, line 138
Handler method for an array of handlers per event type. Per event type (e.g. click, change etc.) this is the one and only handler that is added to the document Object So:
  • All handlers are delegated.
  • For every Event.type there will be exactly one handler, added to the document.
  • metaHandler iterates over the (wrapped) handler lambda's created with the createHandlerForHID factory.
Parameters:
Name Type Description
evt Event the event sent by the browser
