ExtendedNodelistLambdas

ExtendedNodelistLambdas

Source:
  • ExtendedNodeListExtensions.js, line 12
All extension methods for ExtendedNodeList where looping may not be an option. See (the source of) ExtensionHelpers > initializePrototype.

Notes

  • Most methods are chainable. build.cmd
  • [...].css is a collection method, not in this file but also chainable, see the documentation there.
  • (implicit) means the parameter should not be provided in the caller

Example

import $ from "JQueryLike.js";
// chainability means:
$(`<div id="helloworld">`)
 .text(`Example: hello ... world`)
 .append($(`<span> OK</span>`))
 .css({
   marginTop: `0.5rem`,
   border: `3px solid green`,
   padding: `5px`,
   fontSize: `1.2em`,
   display: `inline-block`, })
 .find$(`span`)
 .css({className: `okRed`, color: `red`});
 // result (class names are partly random):
 // <div id="helloworld" class="JQLCreated_YZ5XnwbCEGxH">Example: hello ... world<span class="okRed"> OK</span></div>
