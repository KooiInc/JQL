ExtendedNodelistLambdas

ExtendedNodelistLambdas

Source:
  • ExtendedNodeListExtensions.js, line 10
All extendedNodeListCollectionExtensions for ExtendedNodeList where looping may not be possible. See (the source of) ExtensionHelpers > initializePrototype.
Note: Most methods are chainable.
Note: .style is a collection method, not in this file but also chainable.

Example

// presume $ is the alias for ExtendedNodeList
// chainability means:
$(`<div id="helloworld">`)
  .text(`Example: hello ... world`)
  .styleInline( {
     marginTop: `1rem`,
     border: `3px solid green`,
     padding: `5px`, fontSize: `1.2em`,
     display: `inline-block` } )
  .append($(`<span> OK</span>`))
  .find$(`span`)
  .styleInline({color: `red`});
