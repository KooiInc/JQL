Methods

(inner) changeCssStyleRule(selector, styleValues, cssId)

Source:
  • Styling.js, line 39
change or create some css rule in an existing or dynamically created stylesheet (id: cssId) in the document
Example
// assume changeRuleset is imported as setStyleRule
setStyleRule(".myClass", {color: "#c0c0c0", padding: "0 4px 0 15px"}, "YesItsMyCss");
//           ^ someRule  ^                                             ^
//                       ^ styleProps                                  ^
//                                                                     ^ id of the stylesheet
Parameters:
Name Type Default Description
selector string the (css) selectorText, like ul li.inActive, .someClass etc.
styleValues Object an object with CSSStyleDeclarations
Note: enclose a string value of `content` in quotes (e.g. {content: `'Some string'`})
Note: rule keys should be valid (e.g. {marginRight: `0.3rem`} or {"margin-right": `0.3rem`})
cssId string | undefined customCSS id of the css stylesheet (to create or retrieve), default "customCSS"
