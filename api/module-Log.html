Log

A small module for logging in a fixed positioned log box
Every line logged is preceded by the time it is logged (granularity: milliseconds) The styling of the logbox happens via a <style> element, added to the header of the enclosing document.

(inner) defaultStyling

Source:

defaultStyling is the styling used for a the box used for logging (a HTMLFieldSetElement element). May be overridden by your own styles, but must use the id #logBox.
