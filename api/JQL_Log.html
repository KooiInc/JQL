JQL/Log

Source:
  • Log.js, line 3
Static JQL extension method. A small log library for JQL debugging purposes.

Members

(static) defaultStyling

Source:
  • Log.js, line 18
defaultStyling is the styling used for a the box used for logging (a HTMLFieldSetElement element). May be overridden by your own styles, but must use the id #logBox.
