Convert a camel-cased term to dashed string, e.g. for style rule keys
Example
toDashedNotation(`marginRight`); //=> `margin-right`
toDashedNotation(`borderTopLeftRadius`); //=> `border-top-left-radius`
toDashedNotation(`BorderTopLeftRadius`); //=> `border-top-left-radius`
